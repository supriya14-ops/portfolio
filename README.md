# Supriya B K - Professional Portfolio Website

A sleek, responsive, and beautifully animated single-page portfolio website designed and developed using **HTML5**, **CSS3**, and **React (18.x)**.

## 🌟 Key Features

- **Modern Obsidian Aesthetic**: Dark mode primary design with vibrant Indigo and Cyan accents, translucent glassmorphism cards, and interactive hover effects.
- **Dynamic Greeting**: Interactive Typing Carousel presenting key roles: *Full Stack Developer*, *IISc Research Intern*, and *AI & Web Enthusiast*.
- **Interactive Stack Filter**: Visual grid displaying technical proficiency across Languages, Frameworks, Databases, and Tools, filterable in real-time.
- **IISc Research Internship Timeline**: Detail-oriented career timeline highlighting roles and contributions in the Computational and Data Sciences department at **IISc Bangalore**.
- **Responsive Layout**: Designed mobile-first, ensuring an optimized viewing experience on all screen sizes (mobile, tablet, and desktop) with a slide-out hamburger navigation menu.
- **Light/Dark Theme Switcher**: Header switch button allows users to change themes instantly.
- **Interactive Contact Card**: Easy copying of email (`supriyabk2005@gmail.com`) and phone number (`+91 9019398843`) to clipboard with modern toast confirmations.

---

## 🛠️ Tech Stack

- **Frontend Core**: [React UMD (18.x)](https://react.dev) & [Babel Standalone](https://babeljs.io)
- **Styling & Layout**: Vanilla CSS3 (Custom variables, Flexbox, CSS Grid, Media Queries)
- **Animations**: Keyframes, transition transforms, custom hover scaling, and ambient glows
- **Deployment & Hosting Ready**: Deployed locally & prepped for fast hosting (Vercel, GitHub Pages)

## 🚀 How to Run Locally

You can preview the project locally without installing external Node dependencies by using the built-in Python web server.

### Prerequisites
Make sure Python is installed on your computer.

### Step 1: Open Terminal
Open your terminal (Git Bash, Command Prompt, or VS Code Terminal) and navigate to the project directory:
```bash
cd "C:\Users\supri\.gemini\antigravity\scratch\supriya-portfolio"
```

### Step 2: Start Server
Run the standard Python HTTP server command:
```bash
python -m http.server 3000
```

### Step 3: Open in Browser
Visit the active web link:
**[http://localhost:3000](http://localhost:3000)**
